# APU
Serial Arithmetic Processing Unit (APU) that performs a set of operations on up to two 4-bit binary numbers 
based on a 4-bit operation code (OpCode).\
Split into three parts:\
Data Path Module\
State Generator\
Control Circuit\
Data Path Table:\
